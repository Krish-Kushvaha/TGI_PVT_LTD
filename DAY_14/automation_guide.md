# AI Automation Guide

## Tools
- **Zapier**: No-code automation
- **Make.com**: Visual workflow builder

## Example Workflows
1. Blog post → AI captions → Post to Instagram
2. Form submit → AI reply → Send email
3. RSS feed → AI summary → Slack

## Setup Steps
1. Create account on Zapier/Make.com
2. Pick trigger (e.g., new Gmail)
3. Add ChatGPT action
4. Set output (Slack, Twitter, etc.)
